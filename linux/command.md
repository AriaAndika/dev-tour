# Linux Commands

## File System

- cd, change directory
- ls, list directory
- pwd, print working directory
- mkdir, make directory
- touch, make file
- rm, remove file
- cp, copy file
- mv, move file, can be used to rename
- shred, shred file
- cat, echo file content
- ln, link file
- zip, unzip
- find

## Text

- clear

- awk
- sed
- sort
- less
- head, tail
- cmp, compare file
- diff, what different
- uniq
- tr, replace x with y
